# Assets

- **`svg/`** – Iconos e ilustraciones en SVG.  
  Uso: `SvgPicture.asset('assets/svg/nombre.svg')`.  
  Si Figma avisa "Fix format problem" al exportar, aplica la corrección y vuelve a exportar.

- **`images/`** – Imágenes PNG. Flutter **requiere un archivo en la raíz** (`Omboarding_01.png`, `Omboarding_02.png`) como base. Las variantes van en `1.0x/`, `2.0x/`, `3.0x/`. Si solo tienes variantes, copia una (ej. de `1.0x/`) a la raíz.
